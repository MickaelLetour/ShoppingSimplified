<div> Balise Div

Définition

La balise <div> définit une division ou une section dans un document HTML.
L'élément <div> est souvent utilisé comme conteneur pour d'autres éléments HTML afin de les styler avec CSS ou d'effectuer certaines tâches avec JavaScript.

Exemple

<div style="background-color:lightblue">
<h3>This is a heading</h3>
<p>This is a paragraph.</p>
</div>
